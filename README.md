📌 About
AETHER is a futuristic AI Agent that accepts images, videos, files, and text as input, responds with genius-level answers, and supports voice questioning – all inside an immersive 3D cosmic environment.

✨ Features
🖼️ Image & Video – Upload and analyze visual content

📄 Files – Extract insights from PDF, TXT, JSON, CSV

🎤 Voice Input – Click mic and speak naturally

⌨️ Text Query – Type any question for deep reasoning

🌌 3D Scene – Real-time Three.js with particles, bloom, and glowing core

🧠 Genius Responses – Context-aware, philosophical, creative answers

🛠️ Tech Stack
HTML5 / CSS3 – Glassmorphic UI

JavaScript ES6 – Core logic

Three.js – 3D rendering, effects, particles

Web Speech API – Voice recognition

File API – Multimodal uploads
